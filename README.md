## FRONTEND

```bash
npm install
npm run dev
```

## BACKEND

```bash
cd ./api
python -m venv .venv  
# IF YOU ALREADY HAVE VENV
./.venv/Scripts/activate # For windows
source .venv/bin/activate # For other
pip install -r ./requirements.txt 
flask run
```
